## 🌐 Wave 5 · 从单项目工具进阶为跨项目平台 _(2026-05-02)_

> 5 个项目并行管理时第一次有了"上帝视角"。

| 你看到 | 它解决 |
|---|---|
| 📊 **跨项目数据看板**（侧边栏点"数据看板"）<br/>4 个 KPI（项目数 / 平均膨胀率 / 累计变更 / 累计 +天数）+ 逾期 Top 3 + 标签 Top 5 + 全部项目按膨胀率排行 | 老板问"你这季度管的项目整体怎么样" → 不再只能逐个项目截图给他看 |
| 🎯 **项目目标交付日**（项目顶栏点"设置目标日"）<br/>当天 > 目标日 → 项目名旁红色"已逾期 N 天" chip + Sidebar 红圆点 | 之前的"逾期警告"（W4.8）只对单条变更生效，现在整个项目可以挂红条 |
| 🔗 **只读分享链接**（项目顶栏点"分享只读"）<br/>状态全部塞进 URL fragment（`#share=...`），对方打开就看到一样的项目，但所有编辑按钮都隐藏；可以一键"复制为我的项目"克隆 | "把当前项目状态发给老板看"之前要导 PNG 或 Markdown，现在一条 URL 解决 |
| 🎬 **Demo 项目能演示回放了**<br/>之前回放 dialog 显示"暂无快照"是 bug，现在 demo 也走真实变更流程，5 帧可 scrub | 第一次下载体验的时候不会一脸懵 |
| 📱 **移动端响应式**<br/>< 768px 时侧边栏自动转抽屉（左上 Hamburger 触发），图表横滚 | 出门也能 review，再也不用必须等回电脑 |

## 🛠️ Wave 4 · 数据生命周期 + 标签协作 _(2026-05-02)_

> 把"录入"和"汇报"两端都打通，让数据可流转、可归类、可批量管理。

| 你看到 | 它解决 |
|---|---|
| 📤 **CSV 反向导出**（项目顶栏"CSV 导出"）<br/>当前需求一键导出为 Excel 可读的 CSV | 想在 Excel 里改完批量再导回？现在闭环了 |
| 📁 **5 个项目模板**（新建项目时下拉选）<br/>移动 App 0→1 / SaaS Dashboard / 内部工具 / 数据看板 / E-commerce 后台，每个含 6-10 条预填需求 + 完整依赖链 | 新建项目空白页焦虑，"先填啥"再也不用想 |
| ⏯️ **变更回放动画**（图表区"回放"按钮）<br/>时间轴拖动 / 自动播放 1.1s 一帧 / 空格暂停 / Esc 退出 | 复盘"这个项目什么时候开始膨胀的"特别直观 |
| 🏷️ **变更标签**（记录变更时勾选）<br/>5 个预设：需求方反复 / 技术债 / 上线问题 / 范围扩大 / 其他；列表里每条变更显示标签 chip | 老板问"延期主要因为什么" → 看标签分布 |
| ☑️ **批量删除变更**（变更列表"批量管理"开关）<br/>每行 checkbox + 全选 + 一键删 N 条 | 50+ 变更后清理冗余高效 |
| 🔖 **筛选条件 URL 同步**<br/>搜索词 + 类型筛选 + 角色筛选都同步到 URL，刷新或分享链接保留状态 | "把这条筛选发给同事" 终于能用 |
| 🌒 **暗色模式 polish**<br/>之前暗色下 keyboard help 的按键标签几乎看不见，现在用磨砂玻璃 token 显示 | 暗色不再是"凑合能用"|
| ⚠️ **变更逾期警告**<br/>变更日期 > 项目预计交付 → 红色 ⚠️ 逾期 chip | 录变更日期写错或本身就该结束的项目，第一时间提醒 |

## ⚡ Wave 3 · 力量用户 + 时光机 _(2026-05-02)_

> Linear / Notion 用户期待的全键盘 + 历史回看。

| 你看到 | 它解决 |
|---|---|
| ⌨️ **完整键盘体系**（按 `?` 唤起帮助）<br/>`⌘K` 命令面板 · `⌘⇧C` 记录变更 · `⌘Z` 撤销最近删除 · `g d` 回 demo · `g s` 进设置 · `Esc` 关任意弹窗 | 一只手完成所有操作 |
| 🔒 **焦点 trap + 多步撤销栈**<br/>所有 dialog Tab 不会跑出去；连着删 5 条变更都能逐条撤回 | 误删恐慌缓解 |
| 📥 **批量导入需求**（项目顶栏"批量导入"）<br/>CSV / JSON 双格式，含字段示例，一次性导入数十条 | 从 Excel 表 / 别的项目工具搬数据 |
| 📋 **复制 Markdown 报告**（项目顶栏"复制报告"）<br/>项目状态、变更历史、关键路径全部转成 Markdown 复制到剪贴板，直接贴飞书 / 邮件 | 周报刚需 |
| 🕰️ **时光机历史快照**（项目顶栏"历史"）<br/>每次变更后的项目状态都被保存，可以选任一历史点查看当时的需求列表与图表 | "这个变更影响了哪些"可视化 |
| 🔍 **变更过滤栏**（变更列表上方）<br/>搜描述 / 人名 + 类型 chip + 角色 chip 三维筛选 | 50+ 变更找具体一条 |
| 🖥️ **图表全屏**（图表"全屏"按钮）<br/>填满整个浏览器，Esc 退出 | 客户一起看的时候有 demo 用 |

## 🎨 Wave 2 · UX 深化 + 视觉语言 _(2026-05-01)_

> 从能用到好用：磨砂玻璃 + 拖拽 + 截图证据 + 命令面板。

| 你看到 | 它解决 |
|---|---|
| 🎨 **Apple 风磨砂玻璃 + 暗色模式**<br/>侧边栏、卡片、弹窗全部 frosted glass；右下角主题切换（亮 / 暗 / 跟随系统） | 颜值上对标商业级 |
| 🎯 **膨胀率 hero 卡**<br/>项目顶栏第一张大卡突出"+31%"主视觉 + caption "比原计划多 5 天" | 一眼抓重点 |
| 🖱️ **拖拽排序需求**（按住需求名拖动）<br/>基于 @dnd-kit 的流畅手感 | 调整优先级不再点"上移/下移"|
| 📸 **截图证据**（变更弹窗"贴图"区）<br/>每条变更最多 3 张图（base64 内嵌 IndexedDB），点击放大查看 | 老板说"为啥加了这功能" → 截图甩脸上 |
| 🎯 **⌘K 命令面板**<br/>搜项目 / 需求 / 变更，方向键选 + 回车跳转 | 切项目不再点侧边栏 |
| ⚡ **⌘⇧C 快速记录变更**<br/>不论在哪里都能立即唤出录入弹窗 | 别人找你说"加 2 天" → 立刻记，别忘了 |
| ↩️ **⌘Z 撤销删除**<br/>最近一次删除的变更可以一键复活 | 误删保险 |
| 🪟 **侧边栏项目膨胀 chip**<br/>每个项目名旁显示彩色膨胀率（绿 / 黄 / 红） | 不点进去就知道哪个项目要救 |
| 🚀 **持久化保障升级**<br/>localStorage 双 slot 自动备份 + 空库启动检测 + 一键恢复对话框 | 浏览器清缓存不会丢工作 |
